# BaseNotify 组件

#### 函数调用

- const { openNotify } = useNotify()
- openNotify({ type: 'success', title: 'xxx', message: 'xxxxxxxx' })

## Props

| 名称    | 类型     | 默认值    | 说明   |
| ------- | -------- | --------- | ------ |
| type    | `string` | `success` | `可选` |
| title   | `string` | `成功`    | `可选` |
| message | `string` | `成功`    | `可选` |

默认 内容
